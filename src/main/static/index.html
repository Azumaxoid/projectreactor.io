Build Efficient Reactive Apps and Libraries

Reactor is a foundational library for building non-blocking applications on the JVM. Based on the Reactive Streams Specification, you can power apps that have low tolerance for latency, demand extremely high throughput and make the most of their allocated resources.

Reactor key features

Turbo Message Passing

Reactor Processors, Operators and Timers can sustain high throughput rates on the order of 10's of millions of messages per second. Plus its low memory footprint should go under most of the radars.

Fully Reactive

Reactor is designed to be fully non-blocking with efficient buffers and demand management. Its reactive flow-control implementation leaves not much risk for asynchronous publishing going rogue in uncontrolled overflow situations.

Friendly Adoption

  • As a Reactive Engine/SPI with reactor-core, reactor-net, reactor-aeron. They are exposing reactive streams constructs for focused use cases, eventually combined with Spring, RxJava, Akka Streams, Ratpack and others
  • As a Reactive Consumable API with reactor-stream or reactor-bus which expand the contract to composition and eventing use cases

Guardian of the Latency

An insidious enemy sit between your #CloudReady #distributed #microservices and efficient scalability : Latency. Fortunately, Reactor offers non-blocking and backpressure-ready IO solutions including:

  • Decoding/Encoding : reactor-codec
  • Client/Server for TCP, UDP, HTTP/WS : reactor-net
  • Unicast/Multicast : reactor-aeron

Getting started with Reactor

To use the core components of Reactor like EventBus, Stream, and others, just use the reactor-core artifact. Just replace ${reactorVersion}
in the examples below with the version you want to use (the latest is 2.0.0.RELEASE).

build.gradle pom.xml 
build.gradle
repositories {
  maven { url 'http://repo.spring.io/libs-milestone' }
}

dependencies {
  compile "io.projectreactor:reactor-core:${reactorVersion}"
}
pom.xml
<repositories>
  <repository>
    <id>spring-milestone</id>
    <name>Spring Milestones</name>
    <url>https://repo.spring.io/libs-milestone</url>
  </repository>
</repositories>
<dependencies>
  <dependency>
    <groupId>io.projectreactor</groupId>
    <artifactId>reactor-core</artifactId>
    <version>2.1.0.BUILD-SNAPSHOT</version>
  </dependency>
</dependencies>
